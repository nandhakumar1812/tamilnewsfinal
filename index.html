🚀 Firebase அமைவு வழிகாட்டி

இலவச Firebase Firestore மூலம் உங்கள் செய்திகளை சேமிக்கவும். இது 100% இலவசம் — கிரெடிட் கார்டு தேவையில்லை!

1

Firebase திட்டம் உருவாக்கவும்

→ console.firebase.google.com திறக்கவும்
→ "Create a project" கிளிக் செய்யவும் → திட்டத்திற்கு பெயர் கொடுக்கவும்
→ Google Analytics: "Skip" செய்யவும் → திட்டம் தயார்!

2

Web App சேர்க்கவும் & Config எடுக்கவும்

→ Dashboard-ல் "</>" Web icon கிளிக் செய்யவும்
→ App nickname கொடுத்து "Register app" கிளிக் செய்யவும்
→ firebaseConfig object-ஐ நகலெடுக்கவும்
→ இந்த HTML கோப்பில் FIREBASE_CONFIG-ல் பேஸ்ட் செய்யவும்

3

Firestore Database செயல்படுத்தவும்

→ Left menu: "Build" → "Firestore Database"
→ "Create database" → "Start in test mode" தேர்வு செய்யவும்
→ உங்கள் நிலை (region) தேர்வு செய்யவும் → "Enable" கிளிக்
→ Rules tab: allow read, write: if true; என்று மாற்றி "Publish" செய்யவும்

4

Cloudinary அமைவு (படம் சேமிக்க — இலவசம்)

→ cloudinary.com இல் இலவச கணக்கு திறக்கவும்
→ Dashboard-ல் உங்கள் Cloud Name எடுக்கவும்
→ Settings → Upload → "Add upload preset" → Unsigned → Save
→ Preset name-ஐ நகலெடுக்கவும்
→ HTML கோப்பில் CLOUDINARY_CLOUD & CLOUDINARY_PRESET மாற்றவும்

5

Admin கடவுச்சொல் மாற்றவும் (முக்கியம்!)

HTML கோப்பில் மேலே JavaScript பகுதியில்:
const ADMIN_ID = 'admin'; ← உங்கள் பயனர் பெயர்
const ADMIN_PASS = 'TamilMedia@2025'; ← வலுவான கடவுச்சொல்
